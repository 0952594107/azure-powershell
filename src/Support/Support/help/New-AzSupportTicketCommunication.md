# New-AzSupportTicketCommunication

## SYNOPSIS
Creates a new support ticket communication.

## SYNTAX

### CreateByNameParameterSet (Default)
```
New-AzSupportTicketCommunication -SupportTicketName <String> -Name <String> -Subject <String> -Body <String>
 [-Sender <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### CreateByParentObjectParameterSet
```
New-AzSupportTicketCommunication -SupportTicketObject <PSSupportTicket> -Name <String> -Subject <String>
 -Body <String> [-Sender <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Creates a new support ticket communication.

## EXAMPLES

### Example 1
```powershell
PS C:\> New-AzSupportTicketCommunication -SupportTicketName testticket -Name testmessage -Subject "test subject" -Body "test body"

Id                     : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/providers/Microsoft.Support/supportTickets/testticket/communications/testmessage
Name                   : testmessage
Type                   : Microsoft.Support/communications
CommunicationType      : Web

Sender                 : user@contoso.com
Subject                : test subject
Body                   : test body
CreatedDate            : 1/2/2020 1:15:49 AM
```
