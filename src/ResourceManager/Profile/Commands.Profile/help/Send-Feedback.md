# Send-Feedback

## SYNOPSIS
Sends feedback to the Azure PowerShell team via a set of guided prompts.

## SYNTAX

### Default
```
Send-Feedback
```

## DESCRIPTION
The Send-Feedback cmdlet sends feedback to the Azure PowerShell team.
